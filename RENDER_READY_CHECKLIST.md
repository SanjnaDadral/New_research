## 🆘 Common Issues & Fixes

### Issue: "Build failed"
**Solution:**
1. Check logs for error message
2. Verify all files committed to git
3. Verify Python version in render.yaml

### Issue: "Application Error"
**Solution:**
1. Check web service logs
2. Verify DATABASE_URL is set
3. Run migrations: `python manage.py migrate`

### Issue: "Email not sending"
**Solution:**
1. Verify Gmail app password (not regular password)
2. Check 2FA is enabled
3. Verify EMAIL_HOST_USER and EMAIL_HOST_PASSWORD are set
4. Test with: `python manage.py test_email`

### Issue: "Static files not loading"
**Solution:**
1. Check collectstatic ran in build logs
2. Verify WhiteNoise middleware in settings.py
3. Verify STATIC_ROOT = BASE_DIR / 'staticfiles'

### Issue: "Sessions lost after login"
**Solution:**
1. Verify DATABASE_URL is set
2. Verify session table exists
3. Run: `python manage.py setup_sessions`
