## 🐛 Troubleshooting

### Build Fails: "requirements.txt not found"
- Verify file is in repo root: `ls requirements.txt`
- Check git committed it: `git log --name-status` 

### Build Fails: "No module named X"
- Add missing package to requirements.txt
- Commit and push again

### Static Files 404
- Run: `python manage.py collectstatic --noinput --clear`
- Verify `STATIC_ROOT` in settings.py
- Check Render logs for whitenoise errors

### Database Connection Error
- Verify `DATABASE_URL` env var is set
- Check database is created in Render
- Run: `python manage.py migrate`

### Email Not Sending
- Verify `EMAIL_HOST_PASSWORD` is Gmail App Password (not regular password)
- Check Render logs for SMTP errors
- Whitelist Render IP if using SMTP restrictions

### App Spinning Down (Free Tier)
- Normal behavior - app sleeps after 15 mins
- Upgrade to paid plan for 24/7 uptime
- Use free tier for dev/testing
