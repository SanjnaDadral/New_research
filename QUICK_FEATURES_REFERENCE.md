# 🚀 Quick Reference: Tags, Notes & Citations

## 📌 **One-Minute Summary**

### **🏷️ TAGS** = Organize Papers
```
Add labels like: "AI", "important", "thesis"
Find papers by tag later
```

### **📝 NOTES** = Write Your Thoughts
```
Personal comments on the paper
Ideas, summaries, action items
```

### **📖 CITATIONS** = Copy References
```
APA/MLA/Chicago formats
Click to copy, paste in your paper
```

---

## 🎯 **Quick How-To**

### **Add Tags (10 seconds):**
```
1. View paper result
2. Find "Tags" section
3. Type tag → Click "Add"
4. Done!
```

### **Add Notes (30 seconds):**
```
1. View paper result
2. Find "My Notes" section  
3. Click "Edit"
4. Type notes → Click "Save"
5. Done!
```

### **Copy Citation (5 seconds):**
```
1. View paper result
2. Find "Citation" section
3. Click style (APA/MLA/Chicago)
4. Click "Copy Citation"
5. Paste in your document!
```

## 🔥 **Pro Tips**

### **Tags:**
- Use 3-5 tags per paper (don't over-tag)
- Keep consistent format: `machine-learning` not `Machine Learning`
- Create categories: topics, status, projects

### **Notes:**
- Write notes right after reading
- Include action items: `TODO: Cite in Chapter 2`
- Use bullet points for clarity

### **Citations:**
- Copy immediately (don't forget!)
- Check author names are correct
- Save all citations in one document

**Q: Are tags and notes private?**

**Q: Which citation style to use?**  
A: Check your professor's requirements or journal guidelines.

**Q: Can I search by tags?**  
A: Yes! In library view, filter by tags.
